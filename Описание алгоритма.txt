Шаг 1.
Создание соединения к базе данных через библиотеку psycopg2.
Создание таблицы organization_structure в БД для дальнейшего импорта данных в неё.
Импорт данных из json-файла и вставка записей в таблицу.
Запрос у пользователя идентификатора(id) сотрудника через консоль.

Шаг 2.
Выполнение первого запроса из файла get_office_by_employee.sql,
Запрос при помощи рекурсивного cte находит данные об отделе и офисе искомого сотрудника по id,
далее идёт фильтрация и выбираются только данные (id, name) офиса.
Результат выборки присваивается переменным id_organization и name_organization

Шаг 3.
Выполнение второго запроса из файла get_employees_by_office.sql,
Запрос при помощи рекурсивного cte находит все данные об отделах данного офиса и
всех сотрудниках по id офиса, далее идёт фильтрация и выбираются только данные (name) всех сотрудников данного офиса.
Результат выборки присваивается переменной employees_list.

Шаг 4.
Возвращаем в главный метод названия офиса и список его сотрудников, далее выводим в консоль.
